Add multiple routes using a dict to the API.

Parameters

routes (list of dict) – A list of dictionaries containing routes with the same parameter names as core.API.add_route().

Author

Richard Wen <rrwen.dev@gmail.com>

Example

 7def hello_world2():

 9
10routes = [
